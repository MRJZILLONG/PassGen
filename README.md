# PassGen

PassGen. A password generator

# Requirements
* [Python](https://www.python.org/downloads/release/python-2714/) v2.7
* [Requests](http://docs.python-requests.org/en/master/)

# Download 
```sh
$ git clone https://github.com/Ethical-H4CK3R/PassGen
```

# Install
```sh
$ cd PassGen
$ pip install -r requirements.txt
```

# Execute
```sh
$ python passGen.py
```

# Commands
* *set*: to assign a value
* *exit*:  to terminate script
* *help*:  to display help menu
* *reset*: to reset all of the fields
* *update*: to update to a newer version
* *generate*: to generate a new password list
